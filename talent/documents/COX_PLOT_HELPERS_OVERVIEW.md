# Cox Plot Helpers Overview

`cox_plot_helpers.py` provides plotting utilities used in Cell 11.

## Data Requirements
- Cox plots are built from `df_pipeline_10_cox_ready` (Cell 10) and
  `df_pipeline_11_cox_analysis` (Cell 11).
- Pool metrics must exist before plots (Cells 5–6).

## Competing Risks Bar Plots (CIF Bars)
- `plot_competing_risks_cif_bars()` creates promotion/attrition bar charts.
- Bars are ordered by the binned variable (Q1..Qn) and show final CIF values.
- Optional footnote box under each subplot summarizes bin stats.
- Log scaling for distribution plots is handled in `plot_var_distribution()`.

## Palette and Color Matching
- Bar plots accept hex palettes via plot spec:
  - `cif_bar_palette` (shared)
  - `cif_bar_palette_promo` / `cif_bar_palette_attr` (separate)
- Competing risks line plots use the same palette mapping so line colors match bar colors.
- Gradient palettes are generated by interpolating the provided hex endpoints.

## Plot Spec Flags
- `plot_cif_bars`: enable CIF bar plots for a CR plot spec.
- `cif_bar_show_legend`: toggles bar-plot legend (default off).
- `cif_bar_legend_outside`: legend placement when enabled.
